# Tunneller - Dynamic HTTP Router

Dynamic HTTP router with JavaFX UI for path-based routing to different servers.

## Quick Start

### Run (after Maven is installed)
```bash
mvn javafx:run
```

### Build JAR
```bash
mvn clean package
java -jar target/tunneller-router-1.0.0-shaded.jar
```

## Features
- ✅ Routing Mode: HTTP path-based routing
- ✅ Raw Mode: Direct pipe (100% original performance)
- ✅ JavaFX UI with IntelliJ dark theme
- ✅ Auto-complete for routes
- ✅ Configuration persistence
